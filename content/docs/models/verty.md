Verty is Inserloft's first open-source model, published by [OpenGit](/docs/opengit/introduction).

> Introducing Verty — the first open-source model from Inserloft.

Verty was designed around a clear set of goals:

- **Apache License 2.0** — chosen specifically because it permits commercial use, which drives broader adoption than more restrictive licenses.
- **Open weights** — the full model weights are published, not just a paper or an API.
- **Baby LLM** — Verty belongs to the [Baby LLMs](/docs/models/baby-llms) category: efficient, fast, and small.
- **Developer-oriented** — built with local inference and fine-tuning in mind, not just hosted API access.

Verty currently ships as two variants:

- **Verty Base** — the pretrained foundation model.
- **Verty Instruct** — instruction-tuned for chat and assistant-style use.

Both variants are available for download from the Pixel [Model Registry](/docs/platform/model-registry), along with full [Model Cards](/docs/models/model-cards) and benchmark results.
