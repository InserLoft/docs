OpenGit is Inserloft's open-source team, responsible for everything the company releases publicly.

OpenGit's scope goes beyond models. It also publishes:

- Tools
- Libraries
- SDKs
- Skills for [Cleo](/docs/cleo/skills)
- Standalone programs
- Datasets

[Verty](/docs/models/verty), Inserloft's first open-source model, belongs to OpenGit and was announced as "Introducing Verty" — the first open-source release from Inserloft.
