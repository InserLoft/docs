Agent research explores how models can plan, use tools, and complete multi-step tasks with minimal supervision.

This work directly informs the [Tools](/docs/cleo/tools) and [Skills](/docs/cleo/skills) systems used by Cleo, and is expected to culminate in a dedicated Agents API — see the note on the planned Agents API in [Files API](/docs/apis/files-api).

Key open problems this track focuses on include reliability over long task horizons, safe tool use, and coordinating multiple agents on a shared task.
