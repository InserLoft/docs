This page tracks notable updates across the Platform, Models, and APIs.

- **Verty released** — Inserloft's first open-source model, published under the Apache License 2.0 by [OpenGit](/docs/opengit/introduction).
- **Kyro 3** — current generation frontier model, available through the [Chat API](/docs/apis/chat-api) and [Responses API](/docs/apis/responses-api).
- **Kyro 3.5 (planned, September 2026)** — the next generation of the Kyro frontier family.

For API-level changes specifically, refer to the version notes published alongside each [SDK](/docs/developer/sdks) release.
