A few starting points for common use cases:

- **Chat assistant** — a minimal app using the [Chat API](/docs/apis/chat-api) with a Kyro model.
- **Local inference** — running [Verty](/docs/models/verty) locally for offline or on-device use.
- **Semantic search** — using the [Embeddings API](/docs/apis/embeddings-api) with Verty Base to build retrieval over a document set.
- **Cleo Skill** — a custom [Skill](/docs/cleo/skills) extending Cleo with a domain-specific capability.

Full example repositories are published and maintained through [OpenGit](/docs/opengit/introduction).
