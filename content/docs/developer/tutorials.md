Tutorials walk through complete workflows end to end, rather than isolated code snippets. Recommended starting tutorials:

1. **Your first Project** — create a Project on [Pixel](/docs/platform/pixel), upload a dataset, and publish documentation.
2. **Calling your first model** — authenticate and send a request using the [Responses API](/docs/apis/responses-api).
3. **Deploying a fine-tuned model** — push a model version and create a [Deployment](/docs/platform/deployments).
4. **Building a Cleo Skill** — package a custom capability using the [Skills](/docs/cleo/skills) system.

Each tutorial assumes only the [Getting Started](/docs/getting-started/introduction) material as a prerequisite.
