Beyond generating text, Cleo can use tools to take action — for example, running code, searching the web, or interacting with a Pixel [Project](/docs/platform/projects).

Tools give Cleo the ability to:

- Execute and return the result of code, rather than just describing it
- Look up current information instead of relying only on training data
- Read from and write to resources you've connected, such as documents or Projects

Tool use is what allows prompts like "Cleo, build me a website" to result in actual files and working output, not just a description of how one might build a website.
